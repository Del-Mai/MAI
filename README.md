# MAI

MAI es una aplicación web diseñada para ayudar a los estudiantes a organizar proyectos académicos, gestionar tareas y visualizar fechas de entrega de forma sencilla e intuitiva.

## Características

- Dashboard con resumen de proyectos y tareas.
- Gestión de proyectos.
- Gestión de tareas.
- Búsqueda de proyectos y tareas.
- Calendario dinámico con fechas de entrega.
- Configuración de usuario.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript 

## Ejecución

1. Clonar el repositorio.
2. Abrir el proyecto con Visual Studio Code.
3. Ejecutar `index.html` utilizando Live Server.
